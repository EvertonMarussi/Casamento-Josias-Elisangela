### 🏰 Camponês — O Jogo
## 📖 Sobre o Projeto

**Camponês** é um projeto acadêmico desenvolvido por quatro estudantes com o objetivo de transformar uma narrativa clássica em uma experiência interativa envolvente.

O jogo coloca o jogador no papel de um camponês comum, que precisa enfrentar criaturas sombrias, superar desafios e confrontar o rei tirano para resgatar a princesa e restaurar o equilíbrio do reino.

Este repositório contém o **site oficial do projeto**, desenvolvido com foco em apresentação institucional, narrativa e imersão visual.

---

## 🎯 Objetivo

O projeto tem como principais objetivos:

* Desenvolver uma experiência narrativa interativa
* Aplicar conceitos de desenvolvimento web moderno
* Criar uma identidade visual consistente e imersiva
* Apresentar o jogo de forma profissional e estruturada

---

## 🧩 Estrutura do Site

O site está organizado em seções que guiam o usuário pela experiência do projeto:

### 🔹 Hero Section

Apresentação inicial com vídeo de fundo e chamada principal do jogo.

### 🔹 Motivação

Explica o propósito do projeto e o que motivou sua criação.

### 🔹 História

Uma timeline estruturada em capítulos que apresenta a narrativa do jogo.

### 🔹 Missão

Detalha os objetivos do jogador dentro do universo do jogo.

### 🔹 Bestiário

Apresenta as criaturas e antagonistas, incluindo atributos e níveis de ameaça.

### 🔹 Equipe

Informações sobre os desenvolvedores do projeto.

### 🔹 Contato

Canais para comunicação e acesso ao repositório.



```bash
npm run build
```


## 🎮 Conceito do Jogo

O jogo segue uma progressão baseada em narrativa:

1. Introdução ao reino invadido
2. Enfrentamento de criaturas
3. Evolução do personagem
4. Confronto final com o rei
5. Resgate da princesa

A proposta combina elementos de:

* Aventura
* RPG
* Progressão narrativa
